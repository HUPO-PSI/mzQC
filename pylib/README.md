# MZQC python library

A python library to use mzQC files. Specifically, have a usable object representation of mzQC that can
* serialise
* deserialise
* syntactic checks
* semantic checks

## Install
```
pip install -e git+'https://github.com/HUPO-PSI/mzQC@mzqc-pylib#egg=MZQC-0.0.1&subdirectory=pylib'
```
